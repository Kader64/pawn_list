# Dokumentacja pionków gry

Strona stanowi interaktywną dokumentację wszystkich pionków dostępnych w grze. Została przygotowana jako uzupełnienie głównej dokumentacji projektu, dzięki czemu szczegółowe informacje o pionkach mogą być rozwijane niezależnie od dokumentacji technicznej.

## Zawartość

Każda karta pionka zawiera:

* nazwę pionka,
* grafikę lub model 3D,
* nazwę umiejętności,
* kategorie umiejętności,
* opis działania umiejętności,
* liczbę użyć (jeżeli dotyczy),
* czas odnowienia (jeżeli dotyczy),
* warunek aktywacji (jeżeli dotyczy).

## Cel

Lista pionków będzie rozwijana wraz z postępem prac nad grą. Umieszczenie jej na GitHub Pages pozwala na bieżące aktualizowanie informacji bez konieczności modyfikowania głównej dokumentacji projektu. W dokumentacji technicznej znajduje się jedynie odnośnik do tej strony, dzięki czemu pozostaje ona przejrzysta i łatwiejsza w utrzymaniu.
